####  项目结构

```
uWork
├─applications    # 应用
│  ├─common         # 配置文件
│  │  ├─  script    # 定义好的flask命令（目前包括初始化数据库，快捷创建操作方法文件）
│  │  ├─  tasks     # 功能未知 ？
│  │  ├─  utils     # 组件（如邮件，上传，返回定义，校验等）
│  │  ├─  curd.py   # 数据库 curd 操作，针对 model
│  │  ├─  log.py    # 记录 log 的公共方法
│  │  └─  helper.py # orm 构造器
│  ├─configs      # 配置文件
│  │  └─ config.py  # 配置文件对象
│  ├─extensions  # 注册插件
│  ├─models  # 数据 model 模型
│  ├─schemas # 数据 schemas 类
│  ├─static  # 静态资源文件(暂无)
│  ├─templates  # 静态模板文件(暂无)
│  └─views  # 视图部分(暂无)
├─docs  # 文档说明
├─migrations  # 迁移文件记录（暂未应用，待开发）
├─requirement  # 依赖文件目录（用来安装相关依赖）
└─.env # 项目的配置文件

```
